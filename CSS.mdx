# CSS
웹 페이지의 스타일과 레이아웃을 정의하는 스타일시트 언어이다. 
Cascading Style Sheets의 약자로, HTML 요소의 외관을 설정한다. 
디자인과 레이아웃을 통합 관리한다.

# CSS 셀렉터의 종류와 예시
CSS 셀렉터는 특정 HTML 요소를 선택하여 스타일을 적용한다. 
예시로, 클래스 셀렉터(.className), ID 셀렉터(#idName), 태그 셀렉터(tagName) 등이 있다. 
다양한 조합과 복합 셀렉터도 가능하다.

# 박스 모델(Box Model)
박스 모델은 HTML 요소의 레이아웃을 설명하는 모델이다. 
콘텐츠(content), 패딩(padding), 테두리(border), 마진(margin)으로 구성된다. 
요소의 크기와 배치를 관리한다.

# CSS Flexbox와 Grid의 차이점
Flexbox는 1차원 레이아웃을, Grid는 2차원 레이아웃을 제공한다. 
Flexbox는 주로 아이템을 한 행 또는 열에 정렬하는 데 사용된다. 
Grid는 행과 열을 동시에 배치하는 데 유용하다.

# CSS에서 position 속성의 종류와 차이점
position 속성에는 static, relative, absolute, fixed, sticky가 있다. 
static은 기본 위치, relative는 자신의 위치를 기준으로 이동, absolute는 부모 기준 위치, fixed는 뷰포트 기준, sticky는 스크롤에 따라 변한다.

# CSS에서 float와 clear의 역할
float는 요소를 왼쪽이나 오른쪽으로 배치하여 텍스트나 다른 요소가 주위를 감싼다. 
clear는 float 요소의 영향을 제거하여 다음 요소가 아래로 배치되도록 한다. 
레이아웃 구성에 사용된다.

# CSS 전처리기(Preprocessor)
CSS 코드의 생산성과 유지보수성을 높이는 도구
ex) Sass, Less
변수, 중첩, 믹스인 등을 지원
복잡한 CSS 코드를 더 효율적으로 작성

# 반응형 웹 디자인(Responsive Web Design)
다양한 디바이스와 화면 크기에 맞춰 웹 페이지를 동적으로 조정하는 기술
미디어 쿼리와 유연한 레이아웃을 사용하여 구현
사용자 경험을 향상

# 미디어 쿼리(Media Query)의 역할
미디어 쿼리는 특정 조건에 따라 CSS 스타일을 적용
화면 크기, 해상도, 방향 등에 따라 다른 스타일을 정의
반응형 웹 디자인의 핵심 요소

# CSS 변수 (CSS Custom Properties) 사용 방법
재사용 가능한 값을 정의하고 사용할 수 있는 기능 
--variableName: value; 형식으로 정의 
var(--variableName)로 참조 
코드의 일관성과 유지보수성을 높인다

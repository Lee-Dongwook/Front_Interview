# HTML
HyperText Markup Language의 약자   
웹 페이지를 구조화하고 콘텐츠를 정의하는 마크업 언어   
요소와 태그를 사용하여 텍스트, 이미지, 링크 등을 표시한다.
웹 브라우저는 HTML을 해석하여 웹 페이지를 렌더링한다.

# DOCTYPE
DOCTYPE은 문서 형식을 선언하는 태그로, HTML 문서의 첫 줄에 위치한다. 
브라우저가 문서를 표준 모드로 렌더링하도록 지시한다. 
다양한 HTML 버전에 맞는 문법을 명시한다.

# HTML5의 주요 기능
HTML5는 시맨틱 요소(예: `<article>`, `<section>`), 오디오 및 비디오 태그, 그리고 로컬 저장소 API 등을 도입한다.
웹 애플리케이션을 더 강력하고 상호작용적으로 만든다.
플러그인 없이 멀티미디어를 지원한다.

# 시맨틱 태그
시맨틱 태그는 콘텐츠의 의미를 명확히 하는 HTML 요소이다. 
예를 들어, `<header>`, `<footer>`, `<nav>`, `<article>` 등이 있다. 
검색 엔진 최적화(SEO)와 접근성에 도움을 준다.

# `<div>`와 `<span>`의 차이점
`<div>`는 블록 레벨 요소로, 한 줄 전체를 차지한다. 
`<span>`은 인라인 요소로, 텍스트 줄 내에서 사용된다. 
레이아웃과 스타일링에서의 차이를 반영한다.

# HTML에서의 폼 요소
폼 요소는 사용자 입력을 받기 위한 HTML 태그들이다. 
예를 들어, `<input>`, `<textarea>`, `<button>`, `<select>` 등이 있다. 
데이터를 서버로 제출하기 위한 구조를 제공한다.

# `<meta>` 태그의 용도는 무엇인가요?
`<meta>` 태그는 문서의 메타데이터를 정의한다. 
예를 들어, 문자 인코딩, 뷰포트 설정, SEO 키워드 등을 설정한다. 
페이지의 동작과 검색 엔진 최적화에 영향을 준다.

# `<a>` 태그에서 target="_blank"의 의미는 무엇인가요?
target="_blank"는 링크를 새로운 브라우저 탭이나 창에서 열도록 지정한다. 
사용자 경험을 개선할 수 있지만, 남용 시 사용자 혼란을 초래할 수 있다. 
외부 링크나 파일 다운로드에 주로 사용된다.

# HTML 엔티티
HTML 엔티티는 특수 문자를 HTML에서 표시하기 위한 코드이다. 
예를 들어, &amp;는 앰퍼샌드(&), &lt;는 꺽쇠(`<`)를 나타낸다. 
텍스트 내에서 태그로 인식되지 않도록 한다.

# `<iframe>` 태그의 용도는 무엇인가요?
`<iframe>` 태그는 HTML 문서 내에 다른 HTML 페이지를 삽입한다. 
외부 콘텐츠를 포함하는 데 사용된다. 
예를 들어, 비디오, 지도, 외부 웹사이트 등을 포함할 수 있다.
